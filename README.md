## 기술 스택 (Technique Used) 
### Server(back-end)
 -  Python 
 -  Django
 -  MariaDB, nginx, gunicorn
 
### Front-end
 -  html/css/js 
 -  Bootstrap 5
 -  AOS ( Animate on scroll library )
